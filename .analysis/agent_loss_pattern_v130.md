## L1 60s 지연의 효과 분석

**L1 60s 지연이 차단했어야 할 것**: 이전 봉의 급등이 1분봉 close >= 진입가 조건을 충족할 때만 매수 허용. 

**실제 결과**:

| 건 | 진입 전봉(종가) | 진입봉 | 결과 |
|---|---|---|---|
| DRIFT | 09:51봉 93.1(+4.7%) | 09:52봉 93.3에서 매수 | L1 통과(93.1 >= 진입가 threshold) |
| RAY | 09:07봉 1,224(상승 중) | 09:08봉 1,225에서 매수 | L1 통과(1,224 >= threshold) |
| MANTRA | 09:53봉 15.4(소폭 하락) | 09:54봉 15.4에서 매수 | L1 통과(돌파봉 15.5 이후 15.4 유지) |
| OPEN | 09:25봉 404(소폭 하락) | 09:26봉 404에서 매수 | L1 통과(박스 상단) |
| SOON | 09:29봉 262(하락봉) | 09:30봉 263에서 매수 | L1 통과(직전봉 하락에도 진입) |

**L1이 효과 미흡한 이유**:

1. **L1 조건이 종가 비교이지 모멘텀 소진 감지가 아님**: DRIFT의 경우 09:51봉 +4.7% 급등 종가(93.1)가 WS_BREAK 임계값을 넘어 "1분봉 close >= 진입가"를 충족. 그러나 이미 급등 직후 고점이기 때문에 1분 지연만으로는 부족.

2. **모멘텀 둔화 신호 미감지**: RAY(09:06봉 아래꼬리 -1.6%), SOON(09:29봉 265→262 하락봉), MANTRA(09:53봉 15.5→15.4 소폭 하락)에서 이미 모멘텀 둔화 신호가 있었으나 L1은 이를 차단하지 못함.

3. **거래량 고갈 미감지**: MANTRA 09:55~09:59봉 거래량이 수천 수준으로 고갈됐지만 이미 매수 실행 후.

## 데이터 기반 개선 권고 (최우선 2개)

### 권고 1: 급등봉 연속 확인 2봉 조건 추가 (L1 강화)

**근거**: DRIFT(09:51봉 +4.7%)와 RAY(09:07~09:08봉 연속 상승 +3%)는 L1 1봉 확인으로는 차단 불가. 최고점 1~2봉 이내가 진입점이 됨.

**제안**: L1 조건에 "진입 전봉의 변동폭이 X% 이상이면 2봉 연속 close >= 진입가 확인 필요" 추가.
- 임계값 예시: 진입 직전봉 변동폭 >= 2.5% → 2봉 확인 (60초 추가 대기, 총 L1 = 2분)
- 근거: DRIFT 09:51봉 +4.7%, RAY 09:07봉 +0.6%→09:08봉 +1.2% 연속. 2봉 확인이면 DRIFT의 경우 09:53봉(94.9원, 전봉 93.8 대비 소폭 상승) 확인 후 09:54봉 진입 → 95원 전후 진입으로 손실 감소.
- 단, 오히려 더 비싸게 진입할 수도 있어 조건을 "전봉 변동폭 >= 2.5% AND 2봉 연속 하락 시 차단"으로 구성하는 것이 더 정확함.

### 권고 2: 진입 직전봉 거래량 급감 필터 (돌파 실패 감지)

**근거**: MANTRA는 09:51봉 대량 돌파(2.36M) 후 09:53봉(433K)→09:54봉(202K)→09:55봉(3K)로 거래량이 급감. 이는 돌파 실패의 전형적 신호.

**제안**: OP 스캐너에서 진입 전 1봉의 거래량이 돌파봉 거래량의 X% 미만이면 차단.
- 임계값 예시: 진입 전봉 거래량 < 돌파봉(WS_BREAK 발생봉) 거래량의 20% → 진입 차단
- MANTRA 적용 시: 돌파봉 2,359,503 × 20% = 471,900. 진입 직전봉(09:53) 433,705 < 471,900 → 차단 가능
- 기대 효과: MANTRA 손실 -6,573원 회피. 대신 일부 정상 진입도 차단될 수 있음.

## 종합 결론

1. **V130의 L1 60s 지연은 꼬리매수를 부분적으로만 완화**함. 급등봉 직전에 WS_BREAK가 발생하면 1분 지연 후에도 여전히 최고점 부근에서 진입이 일어남. 5건 중 3건(60%)이 꼬리매수 패턴.

2. **DRIFT -5.84% 초과 손실은 슬리피지**. 10:02봉 단 1분 내 94→90 급락 시 SL 시장가 주문이 대량 매도 쏟아지는 구간에 겹쳐 이론 SL보다 2~3% 낮게 체결. 갭하락 아님.

3. **SL_TIGHT 건들(4건)은 박스권 이탈형**이 2건 포함. 진입 자체가 박스권 상단에서 이뤄지고 장기 진동 후 하락 이탈 구조. SL 수준 조정보다 **진입 품질 필터**(거래량 지속성 확인)가 더 효과적임.
