## 6. 최종 권고

### 권고: minConf 인하 **보류**

**데이터 근거:**
- 6.5~7.0 구간: 유니크 신호 0건 (진입 기회 자체 없음)
- 6.0~7.0 구간: WR 25%, EV -29원/건 → 손실 영역
- 5.5~7.0 구간: WR 29%, EV -7원/건 → 거의 무의미 (수수료 간신히 회수 수준)
- 5.5~6.0 구간: WR 33%, EV +15원/건 → 양수지만 이론상 breakeven 수준 (수수료 0.10% → 최소 WR 50% 필요)

### 리스크 수량화
- minConf 6.0으로 인하 시: 8일 기준 16건 추가, 기대손실 -4.6% (46원/16건)
- minConf 5.5으로 인하 시: 8일 기준 31건 추가, 기대손실 -2.3% (22원)
- **진입 횟수는 늘지만 기대수익은 중립 이하**

### 한계 명시
1. **EMA/MACD Exit 미시뮬**: 실전에서는 두 조건이 추가 조기 청산 발동 → PnL이 이 결과보다 **약간 낮을** 가능성
2. **데이터 기간**: 04-11~04-18 (시장 급락기 포함) — 정상 상승장 데이터 부재
3. **6.5~7.0 신호 0건**: 해당 구간 신호 자체가 드물어 통계 불가
4. **AXL 편중**: 04-16 6건이 KRW-AXL 단일 코인 → 편향 가능성

### 대안 제안
현재 7.0 임계값 유지 + 아래 개선 검토:
- surge_pct 최소값 필터 강화 (현재 vol_gate 기반으로 surge가 낮은 신호 추가 필터링)
- AXL처럼 ema21=DN+ABV(EMA 아래) 신호 차단 강화
- 04-16 KRW-AXL 패턴 분석: 12~15시 매수 → 지속 SL → 시간대 필터 검토
