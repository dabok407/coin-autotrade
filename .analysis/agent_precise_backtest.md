**TOKAMAK +21.9% 격차는 trail 폭과 무관** — peak가 20:39 KST에 나타나는 구조적 한계. OP 스캐너 12:00 만료로 어떤 trail 시나리오도 20:39 peak를 잡을 수 없음.  
**해결책은 trail 변경이 아닌 AD(AllDay) 스캐너 활성화 또는 OP 세션 연장.**

### 핵심 결론

1. **trail 폭이 너무 타이트 (개선 가능)** — V130 ladder에서 peakPct 1.5~3% 구간에 drop 0.5~1.0%, ENSO/XCN/ZBT 등이 2~3% peak 후 일시 하락으로 청산. trail drop 확대(S9처럼 split 30/70 또는 S6 ratchet)로 일부 개선 가능
2. **TOKAMAK은 구조 문제** — peak 20:39 KST는 OP 스캐너 시간 내에 존재하지 않음. trail을 아무리 넓혀도 잡을 수 없음. AD 스캐너 활성화가 유일한 해결책
3. **trail 폭 확대 시 V130-after 22건에서 오히려 손실 증가** — 더 넓은 trail = peak에서 더 많이 하락한 후 청산 = 수익 감소. S3(4%) S4(5%)는 22건 기준 -5~6만원으로 S0 대비 크게 악화
4. **S9(split 30/70)이 최선** — 1차만 30% 매도하고 70%를 더 오래 보유 → 큰추세 케이스에서 2차 수익 극대화, WR 유지
5. **L1 캡은 역효과** — L1 ROI 평균 1.5%인데 캡을 걸면 조기 청산으로 PnL 감소
