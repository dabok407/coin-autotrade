**핵심 차이점**:

1. **코인의 1분봉 단봉 변동성은 주식의 5~20배**  
   → 주식에서 "하루 동안" 오르는 5%를, 코인은 **5~15분 안에** 달성 가능  
   → 반대로 코인은 **5~15분 안에 -5%~-6% 손절**도 발생

2. **코인은 급등 후 급락 패턴이 빈번**  
   → DRIFT: 1분에 +6.3% 후 10분에 -6.4% (V자 후 재붕괴)  
   → 주식은 갭업 후 오전 중 완만한 가격 발견 과정

3. **주식봇 +16% 사례의 배경**  
   → 주식 +16%는 하루 상한가(+30%) 방향으로 5~6시간 지속 상승  
   → Trail이 오전 내내 따라 올라가며 peak를 높임  
   → 주식은 "시간 = 가격 발견의 여유"를 제공

### 4-3. 이론적 동작 차이 분석

**동일한 1.5% trail drop이 주식 vs 코인에 적용될 때**:

```
주식 시나리오:
  - 종목이 +6% peak 도달 후 -1.5% pull back → trail 발동 → +4.5% 익절
  - 1분봉 단봉 변동 0.5~1% 수준 → 1.5% trail은 "2~3봉 조정" 허용
  - 주식은 pull back이 완만해서 trail이 peak를 계속 높일 기회가 있음

코인 시나리오:
  - 종목이 +2.5% peak 도달 후 단봉 -2.3% pull back (1분 안에!) → trail 발동 → +0.2% 청산
  - 1분봉 단봉 변동 1~5% 수준 → 1.5% trail은 "0.5~1봉 조정도 못 견딤"
  - 코인의 일시 pull back이 trail을 즉시 발동 → 보유 지속 불가
```

**V130 실측 결과 (agent_5p_timing_v130.md)**:

- ZBT: peak +11.85% → 코인봇 청산 +1.79% (trail 2.0% 적용)  
- TOKAMAK: peak +21.9% → 코인봇 청산 +2.42%  
- ENSO: peak +10.57% → 코인봇 청산 +1.72%  

**이 수치가 의미하는 것**: trail 2.0%도 코인의 분당 변동성 앞에서 "너무 타이트"하다.  
+2%~3% 시점에서 일시 pull back이 발생하면 trail 즉시 발동 → 보유 포지션 청산 완료.

## Step 5. 코인에서 +5% / +16% 거래가 가능한가?

### 5-1. 코인의 +5% / +16% 가능성

**시장 자체는 충분히 발생한다:**

| 날짜 | 종목 | 당일 peak | 스캐너 포착 여부 |
|------|------|---------|----------------|
| 04-29 | KRW-BLEND | +200% | 포착 → +3.28%에 청산 |
| 04-29 | KRW-XCN | +10.97% | 포착 → +2.56%에 청산 |
| 04-30 | KRW-ZBT | +11.85% | 포착 → +1.79%에 청산 |
| 04-30 | KRW-BIO | +35.97% | 포착 → 청산 기록 (별도 분석) |
| 05-01 | KRW-API3 | +11.78% | 포착 → +1.73%에 청산 |
| 05-01 | KRW-TOKAMAK | +21.9% | 포착 → +2.42%에 청산 |

**결론**: 코인 시장에서 하루 +10~200% 상승은 매주 발생한다. 코인봇도 해당 종목에 진입하고 있다.  
문제는 "시장에 기회가 없어서"가 아니라 **"매도 구조가 기회를 실현하지 못해서"**다.

### 5-2. 주식 +16% vs 코인 +16%의 구조적 차이

```
[주식 +16% 달성 경로]
  시초가 갭업 +6% → 오전 중 완만 추가 상승 → 2~4시간에 걸쳐 +10% 추가
  Trail: 매 1~5분마다 0.1~0.5% peak 갱신 → trail drop 1.5%가 최신 peak 반영
  결과: trail이 6시간 동안 peak를 계속 올리며, pull back 없이 추세 지속 시 +16% 도달 후 장 마감

[코인 +16% 달성 경로 (단일 세션)]
  시초 급등 +16% → 10분 안에 달성 (1분봉 단봉 1~3%씩 쌓임)
  or: 완만한 상승인 척하다가 특정 시점에 단봉 폭등 (BLEND +200%이 1분에 달성된 사례)
  Trail: +2.5% 도달 → 일시 -2.1% pull back → trail 발동 → 나머지 +14% 포기
```

**주식은 "시간이 trail의 친구"**다. 천천히 오르는 동안 trail이 안전하게 peak를 추적한다.  
**코인은 "분단위 변동성이 trail의 적"**이다. 빠르게 오르는 동안 pull back도 분 단위로 크다.

### 5-3. 코인봇이 +5% 거래를 잡으려면

**이론적으로 필요한 trail 폭 계산**:

코인 1분봉 단봉 최대 변동 실측: 1~6% (급등 종목)  
일반적 pull back 크기: 1~3% (1~5분 일시 조정)

따라서 trail drop이 "pull back을 견디면서 추세를 유지"하려면 **최소 3~5%** 필요.

| trail drop | 결과 |
|-----------|------|
| 1.0~2.0% (현재) | 첫 번째 pull back에서 즉시 청산 → peak 2~3% 수준에서 종료 |
| 3.0% | 일반 pull back(1~2%) 1회 허용 → peak 4~6% 가능 |
| 5.0% | pull back(2~4%) 1회 허용 → peak 7~12% 가능 |
| 7.0% | 대형 조정도 허용 → +16% 가능하지만 손실도 심화 |

**trade-off**: trail을 넓히면 더 큰 수익을 잡을 수 있지만, 반대로 SL에 가까워지거나  
수익이 역전되는 리스크가 증가한다. 특히 급등 후 급락(V자 붕괴) 패턴에서 넓은 trail은 치명적.

## Step 6. 권고

### 6-1. 주식봇 전략을 코인에 그대로 적용 가능한가?

**No. 3가지 구조적 이유**:

1. **시간축 불일치**: 주식 +16%는 6시간에 분산. 코인 +16%는 10~30분에 집중.  
   동일 trail 폭에서 주식은 trail이 "천천히 peak를 추적"하지만,  
   코인은 pull back이 trail보다 빠르게 도달.

2. **장 구조 불일치**: 주식은 하루 1사이클, 장 마감 시 강제 청산 없음(익일 보유 가능).  
   코인은 24시간이지만 스캐너 세션 종료가 있고, 갭 없이 연속 가격 움직임.

3. **진입 우위 불일치**: 주식봇은 "갭업" 진입 — 이미 전날 대비 +3~20% 올라서 시작.  
   이 자체가 모멘텀 확인이며 오전 추세 지속을 기대하는 구조.  
   코인봇 OP/MR도 유사하지만, 24시간 거래로 "갭"의 의미가 다름.

### 6-2. 코인 시장에 적합한 매도 구조

**현재 문제의 핵심 (agent_5p_timing_v130.md 결론 인용)**:

```
"진입은 맞았는데 SPLIT_2ND_TRAIL이 botPeak 1.5~3.3% 시점에 청산해버려서,
이후 10~22% 상승 구간을 완전히 놓친다."
```

**권고 매도 구조**:

#### 방안 A: Trail 폭 확대 (가장 직접적)

| 상황 | 현재 | 권고 |
|------|------|------|
| SPLIT_1ST 후 trail drop | 2.0% | peak 3% 미만: 2.0%, peak 3~7%: 3.5%, peak 7%+: 5.0% |
| SPLIT_2ND trail | 2.0% | peak 3% 미만: 2.0%, peak 3~7%: 4.0%, peak 7%+: 6.0% |

> 근거: 코인 1분봉 pull back이 2~4%이므로, trail 2%는 항상 첫 pull back에 매도됨.

#### 방안 B: Breakout 모드 추가 (trail 비활성화 구간)

- SPLIT_1ST 후 "급등 모드" 감지 시 (1분봉 close > 전봉 대비 +2%) trail 일시 중단
- 일정 시간(5~10분) 이후 trail 재활성화
- 장점: 단봉 급등 후 일시 pull back에 청산 방지
- 단점: 구현 복잡도 증가, 급등 후 역전 시 손실 심화

#### 방안 C: QTP 고정 익절 (단순하지만 효과적)

- 현재 OP 스캐너 quickTpPct = 0.70% → **3~5%로 상향**
- Quick TP 도달 시 전량 청산 (trail 없이 고정 익절)
- 코인의 단봉 급등 특성에 적합 (3~5% 달성 가능성 높음)
- 단점: 더 큰 수익(10~20%) 포기

#### 방안 D: Split-Exit 구조 재조정 (권고 핵심)

```
현재:   SPLIT_1ST(40%) → SPLIT_2ND_TRAIL(60%, trail 2%)
권고:   SPLIT_1ST(30%) → SPLIT_2ND_QTP(30%, QTP 고정) → SPLIT_3RD_TRAIL(40%, trail 4~5%)
```

- 1차(30%): 이른 익절로 기본 수익 확보 (1.5~2%)
- 2차(30%): QTP 고정 익절 (3~5%)
- 3차(40%): 넓은 trail로 대형 수익 시도 (trail 4~5%)
- 결과: 대부분 케이스에서 평균 ROI 3~5% 달성, 일부 케이스에서 10%+ 가능

### 6-3. +5% 거래를 일관되게 잡으려면

| 조건 | 현재 상태 | 필요 변경 |
|------|---------|---------|
| SPLIT_2ND trail 폭 | 2.0% | 최소 3.5~5.0% |
| QTP 활성화 | 0.7% (OP에만) | 3~5%로 상향, MR/AD에도 적용 |
| AD 스캐너 활성화 | 4일간 실진입 2건 | 필터 기준(HighConfidence 9.4) 재검토 → 야간 22건 포착 필요 |
| SPLIT_1ST trail drop | 2.0% (peak 2% 미만에서도) | peak 2% 미만: 1.0% (좁게), peak 3%+: 3.0% (넓게) |

**+5% 거래 달성 확률 시뮬레이션 (기존 분석 외삽)**:

- V130 기준: 19건 진입 중 peak 5%+ 도달 0건 (trail 2%에 모두 청산)
- trail 4% 적용 시: 19건 중 추정 5~8건 peak 3%+, 2~4건 peak 5%+ 달성 예상  
  (근거: ZBT +11.85%를 trail 4%로 유지했다면 +7~8% 청산 가능)

## 결론 요약

### 주식봇 +16% 같은 대박 거래가 코인봇에서 가능한 조건

| 번호 | 조건 | 현재 충족 여부 |
|------|------|--------------|
| 1 | 시장 자체 기회 존재 | 충족 (매주 10~200% 종목 발생) |
| 2 | 진입 타이밍 정확 | 충족 (5% 도달 전후 1~11분 이내) |
| 3 | trail 폭이 코인 변동성에 맞음 | **미충족** (trail 2% → 코인 pull back 2~5%에 즉시 청산) |
| 4 | 대형 수익 구간(5~20%) 보유 유지 | **미충족** (SPLIT_2ND_TRAIL이 2~3%에 청산) |
| 5 | AD 스캐너 야간 활성화 | **미충족** (야간 42% 기회 미포착) |

**최종 답**: 코인봇에서도 +5~16% 거래는 가능하다.  
단, **trail drop을 최소 4~5%로 확대**하고 **AD 야간 필터를 완화**해야 현실화된다.  
주식봇 +16%가 나오는 이유는 "주식 시장의 slow 상승이 trail에 유리한 구조" 때문이며,  
코인에서 같은 결과를 얻으려면 trail이 그만큼 넓어야 한다.
